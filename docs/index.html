> 分析完成。`src/renderer` 目录下的 .tsx 文件构成了应用的前端UI。
- layout.tsx: 定义了应用主布局。
- main.tsx: React应用的入口文件。
- GeminiChat.tsx: 核心的聊天界面组件。
- Diff2Html.tsx: 用于渲染代码差异对比。

核心特性

专为提升开发者生产力而设计。

智能聊天交互

基于强大的 Gemini 模型，提供流畅、上下文感知的对话体验，准确理解您的开发意图。

强大的工具集

能够读写文件、执行Shell命令、搜索代码库，将AI的智能与您本地的开发环境无缝结合。

代码理解与生成

内置代码高亮和Diff视图，无论是解释旧代码、生成新模块还是审查变更，都直观高效。

跨平台桌面应用

基于 Electron 和 React 构建，提供在 Windows, macOS 和 Linux 上一致的、原生的使用体验。

灵活的进程管理

采用多进程架构，确保AI任务在独立的Worker中运行，保持UI流畅响应，不阻塞您的操作。

100% 开源

代码完全开放，由社区驱动。欢迎您贡献代码、提出建议，共同打造更强大的AI开发工具。
